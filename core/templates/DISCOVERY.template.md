# Discovery Checklist

## Instructions for AI Agents

When adding discovery questions:
- Use checkbox format: `- [ ]` for unanswered, `- [x]` for answered
- Include metadata: Ask, Check, Status, Priority
- Mark as answered when information is found
- Update "Last Updated" date when changes are made

**Last Updated:** [DATE]

---

## Example Question Format

- [ ] **[Question text]**  
  - **Ask:** [Who would know the answer]
  - **Check:** [Where to look - specific emails, documents, or people]
  - **Status:** Unknown / Partial / Answered
  - **Priority:** High / Medium / Low
  - **Answer:** [If answered, include the answer here]
  - **Source:** [If answered, cite the source]
